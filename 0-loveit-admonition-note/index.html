admonition(警告/訓詞) shortcode 提供 12 種樣式讓你可以在文章中放入各種提示

Markdown 或 HTML 也都可以寫入 admonition

Note
Some note tips…
1
2
3

    Some note tips...
  {{< /admonition >}}

admonition 有三個參數可以設定，你可以選擇以下兩種寫法

1
2
3

{{< admonition type=tip title="This is a tip" open=false >}}

{{< /admonition >}}

OR

1
2
3

{{< admonition tip "This is a tip" false >}}

{{< /admonition >}}
